## Backend

**Firebase Firestore** (Google Firebase Cloud Database)

Real-time NoSQL database hosted on Google Cloud. Data tersimpan di cloud dan dapat diakses kapanpun.

## Main Features (5 Fitur Utama)

| #   | Fitur                         | Deskripsi                                                                                               |
| --- | ----------------------------- | ------------------------------------------------------------------------------------------------------- |
| 1   | 📝 **Notes (Pencatatan)**     | Buat, edit, hapus catatan dengan kategori & warna. Tersimpan di Firebase Firestore secara real-time.    |
| 2   | ✅ **Task Manager**           | Kelola daftar tugas dengan prioritas (High/Medium/Low) dan deadline. Progress tracker terintegrasi.     |
| 3   | ⏰ **Focus Timer (Pomodoro)** | Timer fokus belajar dengan teknik Pomodoro (25/5 menit). Dilengkapi notifikasi suara dan tracking sesi. |
| 4   | 📅 **Study Planner**          | Jadwal belajar mingguan. Tambah sesi belajar per hari, lihat dalam tampilan grid per minggu.            |
| 5   | 📊 **AI Chat**                | Untuk melakukan tanya jawab dengan AI                                                                   |

## Tech Stack

- **Frontend**: React 18 + Vite
- **Styling**: Tailwind CSS v3
- **Backend**: Firebase Firestore
- **Icons**: Lucide React
- **Deploy**: Vercel

## Getting Started

```bash
# Install dependencies
npm install

# Setup .env
cp .env.example .env

# Run development server
npm run dev

# Build for production
npm run build
```
